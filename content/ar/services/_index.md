خدمات شاملة في الخصوبة والصحة الإنجابية.
{.display-6}

{{% /blocks/cover %}}

{{% blocks/lead color="white" %}}

نقدم مجموعة كاملة من علاجات الخصوبة وخدمات أمراض النساء،
باستخدام أحدث التقنيات والمناهج القائمة على الأدلة العلمية.

{{% /blocks/lead %}}

{{% blocks/section color="primary" type="row" %}}

{{% blocks/feature title="أطفال الأنابيب (IVF)" icon="fa-solid fa-flask" %}}
يستخدم برنامج أطفال الأنابيب لدينا أحدث تقنيات المختبر لتحقيق أعلى
نسب النجاح. يتم مراقبة كل دورة وتخصيصها بعناية.
{{% /blocks/feature %}}

{{% blocks/feature title="الحقن المجهري (ICSI)" icon="fa-solid fa-microscope" %}}
تقنية متقدمة للتلاعب المجهري يتم فيها حقن حيوان منوي واحد مباشرة
في البويضة، مثالية لحالات عقم الذكور.
{{% /blocks/feature %}}

{{% blocks/feature title="التلقيح داخل الرحم (IUI)" icon="fa-solid fa-syringe" %}}
علاج خصوبة أقل توغلاً يتم فيه وضع الحيوانات المنوية المحضرة مباشرة
في الرحم أثناء فترة الإباضة.
{{% /blocks/feature %}}

{{% /blocks/section %}}

{{% blocks/section color="white" type="row" %}}

{{% blocks/feature title="تقييم الخصوبة" icon="fa-solid fa-clipboard-check" %}}
تقييم شامل يشمل الفحوصات الهرمونية والموجات فوق الصوتية
وتحليل السائل المنوي والإجراءات التشخيصية المتقدمة.
{{% /blocks/feature %}}

{{% blocks/feature title="تجميد البويضات والأجنة" icon="fa-solid fa-snowflake" %}}
أحدث تقنيات التجميد السريع للحفاظ على الخصوبة، مما يمنحكم
المرونة في تخطيط عائلتكم وفقاً لجدولكم الزمني.
{{% /blocks/feature %}}

{{% blocks/feature title="أمراض النساء والتوليد" icon="fa-solid fa-heart-pulse" %}}
خدمات صحة المرأة الشاملة بما في ذلك الرعاية الروتينية ومتابعة
الحمل وجراحات النساء طفيفة التوغل.
{{% /blocks/feature %}}

{{% /blocks/section %}}

{{% blocks/section color="primary" type="row" %}}

{{% blocks/feature title="خصوبة الرجال" icon="fa-solid fa-user-doctor" %}}
تقييم وعلاج متخصص لعقم الذكور، بما في ذلك تقنيات
استخلاص الحيوانات المنوية جراحياً (TESA/MESA).
{{% /blocks/feature %}}

{{% blocks/feature title="الفحوصات الجينية" icon="fa-solid fa-dna" %}}
فحص جيني قبل الزرع (PGT) لفحص الأجنة للكشف عن
الحالات الوراثية قبل النقل.
{{% /blocks/feature %}}

{{% blocks/feature title="الاستشارة والدعم" icon="fa-solid fa-hand-holding-heart" %}}
خدمات الدعم النفسي والاستشارة لمرافقتكم خلال رحلة
الخصوبة بثقة واطمئنان.
{{% /blocks/feature %}}
